Your name will be automatically added here when:

## Recognition Categories

We recognize different types of contributions:

- 💻 **Code** - Writing code
- 📖 **Documentation** - Writing or improving docs
- 🐛 **Bug reports** - Filing detailed bug reports
- 💡 **Ideas** - Contributing to design discussions
- 🔍 **Review** - Code review and feedback
- 🧪 **Testing** - Testing releases and providing feedback
- 🎨 **Design** - UI/UX design work
- 📢 **Evangelism** - Spreading the word about re_gent
- 💬 **Community** - Answering questions, helping others
